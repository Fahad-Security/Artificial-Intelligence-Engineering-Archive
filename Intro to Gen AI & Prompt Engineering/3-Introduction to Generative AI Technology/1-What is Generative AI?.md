# What is Generative AI?

Generative AI is an advanced type of **Artificial Intelligence (AI)** that can create entirely new content instead of only analyzing existing data.

It can generate:

- Text
- Images
- Videos
- Audio
- Source code
- Digital simulations

# The Difference Between AI, Machine Learning, and Generative AI

## 1. Artificial Intelligence (AI)

AI is the broad field focused on making computers imitate human intelligence.

### Examples:
- Voice assistants  
- Chatbots  
- Recommendation systems  

---

## 2. Machine Learning (ML)

Machine Learning is a branch of AI where systems learn from data instead of being manually programmed.

### Examples:
- Spam detection  
- Price prediction  
- Image analysis  

Traditional ML mainly focuses on:
- Classification
- Prediction

---

## 3. Generative AI

Generative AI is the next evolution of AI systems.

Instead of simply recognizing a cat image, it can create an entirely new cat image that never existed before.

Instead of only analyzing text, it can generate:
- Articles  
- Code  
- Stories  
- Summaries  

---

# How Generative AI Works

Generative AI is built on technologies such as:

- Deep Learning  
- Neural Networks  
- Transformers  

Especially the **Transformer Architecture**.

### Popular Models:
- GPT  
- BERT  

---

# How Are These Models Trained?

Models are trained using:
- Billions of words  
- Massive image datasets  
- Large-scale internet data  

During training, the model learns:
- Patterns  
- Relationships between words  
- How to generate logical sentences and realistic outputs  

---

Classifying posts as:
