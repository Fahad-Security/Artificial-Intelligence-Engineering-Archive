# Deep Learning — A Simple Overview

## What is Deep Learning?

Deep Learning is a branch of **Machine Learning**, which itself is a branch of **Artificial Intelligence (AI)**.

- **Artificial Intelligence (AI):** Enables machines to mimic human intelligence.
- **Machine Learning (ML):** Allows machines to learn from data using algorithms.
- **Deep Learning (DL):** An advanced form of machine learning based on neural networks inspired by the human brain.

---

# How is Deep Learning Different from Machine Learning?

In traditional machine learning, humans must manually define the important features for the system.

### Example:
To distinguish between a tomato and a cherry, we might provide features such as:

- Size
- Stem shape

In deep learning, neural networks automatically discover these features without human intervention. However, this requires a very large amount of data.

---

# How Neural Networks Work

A neural network consists of:

1. **Input Layer**
2. **Hidden Layers**
3. **Output Layer**

### Example: Handwritten Digit Recognition

- An image may contain `28 × 28` pixels = `784` inputs
- Each pixel sends information into the neural network
- The system calculates:
  - **Weights**
  - **Biases**
- The result passes through an **Activation Function**
- Finally, the network predicts the correct digit

# Challenges of Deep Learning

## 1. Requires Massive Amounts of Data

The more data available, the better the performance.

## 2. Requires High Computational Power

Deep learning commonly relies on powerful GPUs for training.

## 3. Long Training Time

Training models can take hours, days, or even months.

# Core Idea

Deep learning aims to make computers "understand" and learn in a way similar to the human brain.

It powers many modern technologies, including:

- Real-time translation
- Image recognition
- Smart assistants
- Autonomous vehicles
- Medical data analysis

Deep learning is still evolving, and its future potential is enormous.
